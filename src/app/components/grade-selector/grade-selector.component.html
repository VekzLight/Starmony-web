El Código de la progresión debera tener cada grado ( I, II, III, IV, V, VI, VII ) separado por el símbolo "-".

Cada grado debera tener el tipo de acorde justo después del grado. (ejemplo: Im-II7)

De no se especificar un tipo de acorde se asumirá que es mayor.

Puede seleccionar el tipo de acorde con el selector de acorde.
